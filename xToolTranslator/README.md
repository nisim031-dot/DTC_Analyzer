# xTool מתרגם — מדריך מתומצת

## מה זה
אפליקציית Android שמרחפת מעל אפליקציית הסורק (D7BT) כבועה צפה 🌐,
לוכדת את המסך, מזהה טקסט (OCR) ומתרגמת אנגלית→עברית בזמן אמת.
כוללת גם שירות נגישות שקורא תפריטים/כפתורים ישירות.

---

## התקנה (הדרך שעובדת — בלי adb)
1. הורד את ה-ZIP שנשלח בצ'אט → העבר לטאבלט (USB → "העברת קבצים", לתיקיית Download).
2. בטאבלט: מנהל קבצים → **לחיצה ארוכה** על ה-ZIP → **Extract / חלץ**.
3. לחץ על `xtool_translator.apk` שנוצר → אשר "אפשר ממקור זה" → **התקן**.
4. פתח את "xTool מתרגם" → אשר 3 הרשאות:
   א. הצגה מעל אפליקציות (Overlay)
   ב. נגישות → הפעל "xTool מתרגם"
   ג. "התחל לכידת מסך?" → אשר

> פעם ראשונה צריך אינטרנט (ML Kit מוריד מודל עברית ~30MB). אח"כ אופליין.

## פתרון תקלות
| תקלה | פתרון |
|------|-------|
| הצ'אט לא נותן להוריד APK | מקבלים כ-ZIP, מחלצים בטאבלט |
| "האפליקציה לא הותקנה" | בעיית מעבד — צריך APK עם 32+64 ביט (הגרסה 50MB) |
| אין "אפשרויות מפתחים" | הטאבלט נעול — לא צריך אותן, מתקינים מקובץ |
| `adb` לא מזוהה ב-PowerShell | להריץ עם `.\adb` (נקודה-קו לפני) |
| מנהל קבצים נותן רק "Open as" | זה ZIP — לחיצה ארוכה → Extract, לא לחיצה רגילה |

---

## גרסאות הקובץ (לזיהוי)
- 109MB = כל המעבדים (universal)
- 33MB = 64-ביט בלבד (נכשל בטאבלט הזה)
- **50MB = 32+64 ביט (הנכון לטאבלט שלך)** ← זה מה שצריך

---

## פרטים טכניים (לבנייה מחדש)
- פרויקט: `xToolTranslator/` ‏(Android, Kotlin, minSdk 26 / Android 8)
- ספריות: ML Kit `text-recognition` + `translate`
- בנייה: `gradle assembleDebug` → `app/build/outputs/apk/debug/`
- ABI: `armeabi-v7a` + `arm64-v8a` (ב-`app/build.gradle.kts`)
- קבצי קוד עיקריים:
  - `MainActivity.kt` — בקשת הרשאות + הפעלת השירות
  - `LiveTranslateService.kt` — לכידת מסך + OCR + תרגום
  - `FloatingBubble.kt` — הבועה הצפה
  - `HybridReaderService.kt` — נגישות (קריאת תפריטים)
  - `TranslationEngine.kt` — מנוע תרגום אופליין

## שלבים הבאים (אופציונלי)
- דיוק זיהוי קודי DTC (עמודות DTC Code / Trouble Code) במקום תרגום מילולי
- חיבור ל-KNOWLEDGE_BASE מ-DTC Analyzer → הסבר/חומרה/פעולות בעברית
- אייקון אמיתי לאפליקציה
- צבע בועה לפי חומרה (אדום/צהוב/ירוק)
